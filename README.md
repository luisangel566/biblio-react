# 📚 Biblioteca Personal - Frontend

Aplicación web desarrollada con **React + Vite** para gestionar una biblioteca personal. Permite visualizar, agregar y organizar libros de forma sencilla mediante una interfaz moderna.

## 🚀 Tecnologías
- React
- Vite
- JavaScript
- CSS

## ✨ Funcionalidades
- 📖 Visualización de libros
- ➕ Registro de nuevos libros
- 🔍 Búsqueda de libros
- 🗂 Organización de información

## 📂 Estructura del Proyecto
src/
components/
pages/
services/
App.jsx
main.jsx
